This webpage and GitHub repository are supplements to the paper "Incentive Analysis of Hybrid PoW/PoS Casper the Friendly Finality Gadget Using PRESTO". The buttons below can be used to produce the numerical results used to create Figures 3 and 4 in the paper.
Above all, the underlying code (which can be found at https://github.com/casper-paper/casper-paper.github.io) has been made publicly available to make the paper fully reproducible, and to aid researchers who wish to extend or modify the experiments.
